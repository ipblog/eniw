Batch Export Patents

Rate this Content 0
     
0 Votes

 

User can export the download data or files out of the system.

1. User can export the download patents's bibliographic datas to a .xls excel file use the export to excel command in the Grid Tool bar.

2. User can export the download patents's bibliographic datas to a .xls excel file use the export to excel (Advanced) button in the Grid Tool bar.

Different from 1, User can choose the export fields when use export to excel (Advanced) function.

when use export to excel (Advanced) function, user need have installed Office excel.

3. User can export the fulltext pdf files use the Export command  in the Grid Tool bar.

 User can change the exported file name int the export form. user can use App NO/Pub NO/Patent NO/ or patent title as the exported pdf file name.
